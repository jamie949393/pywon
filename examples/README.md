# Pywon Example Programs

This directory contains example programs to help you learn Pywon.

## Running Examples

```bash
pywon examples/hello.pywon
```

Or if pywon is in the current directory:

```bash
./pywon examples/hello.pywon
```

## Available Examples

### hello.pywon
The classic "Hello World" program. A simple introduction to Pywon.

**Concepts:** `print`

### calculator.pywon
A basic calculator that performs addition, subtraction, multiplication, and division.

**Concepts:** `input`, `print`, math operations (`add`, `sub`, `multi`, `divi`)

### guessing_game.pywon
A number guessing game where you try to guess the secret number.

**Concepts:** `input`, `if`, conditionals, comparisons

### countdown.pywon
Counts down from 10 to 1, then prints "Blast off!"

**Concepts:** `loop`, `set`, `sub`, variables in loops

### age_checker.pywon
Determines your age category (child, teenager, adult, senior) based on your age.

**Concepts:** nested `if` statements, multiple conditions

### multiplication_table.pywon
Generates a multiplication table for any number you enter.

**Concepts:** `loop`, math operations, counters

### greeting.pywon
Creates a personalized greeting based on your name, favorite color, and age.

**Concepts:** multiple `input` statements, string variables, `if/else`

## Learning Path

1. Start with `hello.pywon` - Learn basic output
2. Try `greeting.pywon` - Learn input and variables
3. Run `calculator.pywon` - Learn math operations
4. Play `guessing_game.pywon` - Learn conditionals
5. Watch `countdown.pywon` - Learn loops
6. Explore `age_checker.pywon` - Learn nested conditionals
7. Generate `multiplication_table.pywon` - Combine loops and math

## Modifying Examples

Feel free to modify these examples! Try:
- Changing the numbers
- Adding more conditions
- Creating your own variations
- Combining concepts from multiple examples

## Creating Your Own

Use these examples as templates for your own programs. The syntax is simple and easy to understand.
